# JElecAccounts ⚡

JElecAccounts is a mobile-responsive, private management dashboard designed for electrical shop owners to efficiently track daily financial accounts, inventory, and supplier transactions.

## 🚀 Features

- **Daily Ledger:** Track daily sales, operational costs, and real-time profit calculations.
- **Inventory Management:** Categorized stock tracking for items like fans, LED bulbs, and electrical fittings.
- **Supplier & Expense Logs:** Keep a history of supplier bills and general shop expenses.
- **Real-Time Sync:** Powered by Supabase to ensure data is updated instantly across all devices.
- **Secure Access:** Private management-only portal to ensure financial data stays confidential.

## 🛠️ Tech Stack

- **Frontend:** React.js, Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Backend/Database:** Supabase (PostgreSQL + Real-time)
